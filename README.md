# 🧠 CodeDoctor

CodeDoctor is an AI-powered code assistant that helps developers **fix, review, and improve code instantly** using modern LLMs.

Built with a clean developer-focused UI, it combines a powerful code editor with intelligent AI feedback — making debugging and optimization faster than ever.

---

## ✨ Features

* ⚡ **Fix Code Instantly**
  Automatically detects and fixes bugs in your code.

* 🔍 **Code Review (AI Powered)**
  Get structured feedback like a senior developer:

  * Code quality rating
  * Issues & bugs
  * Improvements & optimizations

* 🧠 **Smart AI Responses**

  * Clean formatted output
  * Markdown rendering
  * Syntax-highlighted code blocks

* 💻 **Multi-language Support**

  * JavaScript
  * Python
  * Java
  * C++

* 🎨 **Modern UI**

  * Monaco Editor (VS Code-like experience)
  * Dark theme
  * Smooth UX

---

## 🛠 Tech Stack

* **Frontend:** React + Vite
* **Editor:** Monaco Editor
* **AI API:** Groq (OpenAI-compatible)
* **Styling:** Tailwind CSS
* **Markdown Rendering:** react-markdown
* **Syntax Highlighting:** react-syntax-highlighter

---

## 📦 Installation

```bash
# Clone the repo
git clone https://github.com/avishjhalani/CodeDocotor.git

# Go into the project
cd CodeDocotor

# Install dependencies
npm install

# Start the app
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the root:

```env
VITE_GROQ_API_KEY=your_api_key_here
```

---

## 🚀 Usage

1. Select a programming language
2. Write or paste your code
3. Click:

   * **Fix Code** → to auto-correct errors
   * **Review** → to get detailed feedback
